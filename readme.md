## Social-security number verification
<hr>

This app would allow user to "log in" if correct social-security number is entered. The Luhn algorithm is not used in this example.

Allowed formats:
* YYYYMMDD-XXXX 
*  YYYYMMDDXXXX
*  YYMMDD-XXXX
*  YYMMDDXXXX

<hr>

### Features:
* Regex
* Date checker
* Vue
